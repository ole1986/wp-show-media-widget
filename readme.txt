Display media images or pdf documents in a widget filtered by categories

== Description ==

This plugin is used to display thumbnail images (pdf supported) in a widget.
The main difference from other media widgets, is the use of taxonomy shipped with the [Enhanced Media Library](https://wordpress.org/plugins/enhanced-media-library) plugin

**Features**

* filter by categories (using Enhanced Media Lib)
* limit the output of thumbnails
* "show more" link to load additional thumbnails
* open in new tab
* hide / show media title


== Installation ==
Add it through wordpress or unpack the downloaded zip file into your wp-content/plugins directory
